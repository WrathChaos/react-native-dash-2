# Installation

Add the dependency:

```bash
npm i react-native-dash-2
```

## Peer Dependencies

Zero Dependency

# Usage

## Import

```jsx
import Dash from "react-native-dash-2";
```

## Horizontal Usage

```jsx
<Dash style={{ width: 100, height: 1 }} />
```

## Vertical Usage

```jsx
<Dash style={{ width: 1, height: 100, flexDirection: "column" }} />
```
